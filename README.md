# Конкурентная таблица

Программа создана для упрощения работы сотрудников отдела закупок.
Автоматизирует рутинный процесс сбора коммерческих предложений из PDF-счетов
и формирования единой конкурентной таблицы для сравнения поставщиков.

## Возможности

- Графический интерфейс на CustomTkinter
- Выбор папки с PDF и фильтрация счетов по чекбоксам
- Автоматическое распознавание таблиц в PDF (camelot)
- Заполнение шаблона конкурентной таблицы
- Формулы (ц/ед = сумма / кол-во, SUM-итоги) и форматирование
- Повторный запуск — новые поставщики добавляются справа

## Архитектура

| Модуль | Назначение |
|--------|-----------|
| `recog.py` | Ядро: извлечение таблиц из PDF, заполнение шаблона Excel |
| `normalizer.py` | Нормализация колонок (товар, кол-во, сумма, без НДС) |
| `gui.py` | Графический интерфейс на CustomTkinter |

**Стек:** Python 3.13+ → camelot-py / Ghostscript (PDF) → pandas (данные) → openpyxl (Excel) → CustomTkinter (GUI)

**Шаблон** `konkurent.xlsx`: 2 блока поставщиков по 6 колонок
(предложено, кол-во, ед.изм, ц/ед, сумма без НДС, сумма с НДС),
мета-строки (договор, сроки поставки, доставка, условия оплаты, комментарий),
строка «Сумма» с SUM-формулами.

**Поток данных:**
```
PDF-счет → camelot (lattice/stream) → нормализация колонок
→ pandas DataFrame → openpyxl → конкурент <папка>.xlsx
```

## Требования

- **Python 3.13+**
- **Ghostscript** (`gs` на Linux, `gswin64c` на Windows) — [скачать](https://ghostscript.com)

## Установка и запуск

```bash
git clone https://github.com/Itanik/konkurent.git
cd konkurent
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/python3 gui.py
```

### CLI-режим

```bash
.venv/bin/python3 recog.py /путь/к/папке
```

## Использование

1. Выберите папку с PDF-счетами (кнопка **«Обзор...»**)
2. Отметьте нужные счета (по умолчанию включены все)
3. Нажмите **«Запустить»**
4. По готовности — нажмите **«Открыть результат»**

Имя выходного файла: `конкурент <первые 2 слова папки>.xlsx`

## Portable-сборка

```bash
# Linux
./build_portable.sh
# → dist/конкурент/конкурент (~285 МБ)

# Windows
build_portable.bat
# → dist\kongkurent\kongkurent.exe (~300 МБ)
```

На целевой машине должен быть установлен Ghostscript.

Готовые сборки автоматически собираются в **GitHub Actions**:
`https://github.com/Itanik/konkurent/actions` → последний запуск → **Artifacts**
